# About the project
Block.Note is simple CLI text editor written in C++. This program is available for both Linux, Windows and Irion OS.

# Download and Launch
``` bash
# Clone the repository
git clone https://github.com/filcherock/block.note.git
cd block.note

# Launch
sh build.sh
```
